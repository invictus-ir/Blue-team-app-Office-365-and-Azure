Before you begin:
This app contains one macro that is used to define which index will be searched. 
Please update this macro to reflect the index, the macro is called `ual` and can be found Settings--> Advanced search settings --> Search Macros 

Dependencies:
- One visualisation uses a custom timeline visualisation to use this please install. Timeline app https://splunkbase.splunk.com/app/3120/
